HTML TAGS & ATTRIBUTES

  1. < ! DOCTYPE html >
    2. This is use to tell the browser it is a HTML5 FILE
  2. < html >....< / html >
    3. This tag is a sign of html element parts are started and its also called as a root tag.
  3. < head >....< / head >
    4. This tag is a brain part of a html it is also called as container and It is generally use for a difining elements are like style,title and metadata etc.
  4. < title >....< / title >
    5. We can use this as a tag and attribute.Genrally it is use for giving the page title inside the head tag and use as a giving a tooltip for users.
  5. < body >....< / body >
    6. This tag is a visible part of the html document file.All the display content is written in the body tag with the help of differnt elements.
  6. < h1 >....< / h1 >
    7. This is a heading tag, Heading tag is start from h1 and ends at h6, h1 is the most important and h6 is the least important. NOTE: bydefault browsers added white spaces to it.
  7. < p >....< / p>
    8. p denotes to paragraph and this tag is use for a paragraph. NOTE: bydefault browsers added white spaces to it.
  8. < i >....< / i >
    9. This is a text formatting tag, i denotes to italic.
  9. < u >....< / u >
    10. U stands for underline. This tag is used for underline the text.
  10. < b >....< / b >
    11. B stands for bold. This tag is use for bold the text.
  11. < br >
    12. This is a empty tag, empty tag is not necessary to close we can close it like < br / >. Br stands for brakeline. Browser leaves the extra spaces and new line for linebrake this tag is used.
  12. < hr >
    13. This is also a empty tag. HR stands for horizontal line. It is used for draw a horizontal line or seprate the contents.
